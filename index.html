Clinical Interpretation of Diastolic Function in Suspected Heart Failure

Bedside documentation & decision-justification framework for Acute Medicine, Geriatrics & General Internal Medicine. Integrates natriuretic peptides with echocardiographic context to support structured interpretation without assigning diastolic grade or a definitive diagnostic label.

Not a HFpEF diagnostic tool and not a diastolic grading tool; use within a wider guideline-based pathway.

Purpose & scope

Supports structured interpretation and documentation of echocardiographic diastolic-relevant context and natriuretic peptides in patients with suspected heart failure, particularly where ejection fraction is preserved or mildly reduced.

This tool: • Does NOT assign diastolic grade • Does NOT mandate treatment • Does NOT replace specialist cardiology assessment • Helps organise clinical + echo + NP evidence into copy-ready documentation

Module 3 — Structural echo features (supportive)

Reference: normal ≤34 ml/m².

LV wall thickness

Pulmonary pressures (optional)

Supportive threshold: TR Vmax >2.8 m/s.

Supportive threshold: PASP >35 mmHg.

Optional Doppler context (documentation; supportive only when interpretable)
Doppler values are recorded when available. They are never used to assign diastolic grade. If any non-interpretable condition is present (AF, significant MR, mod/sev AS, limited Doppler, pacing/fusion), Doppler values are displayed for documentation only and do not contribute to supportive logic.
When interpretable, Doppler supportive context is treated cautiously: E/e′ >14 and/or low e′ (septal <7, lateral <10) can strengthen supportive wording, but cannot independently create a “HFpEF supportive” output, cannot override NP-low, and cannot override valve-dominant safeguards.
Safety principle

Normal resting echo surrogates do not exclude exertional or load-dependent HFpEF.

Resting echo can be neutral; clinical course and response matter.

Input safety

Non-physiological values (≤0) are ignored. Implausible values are flagged and ignored to reduce documentation errors.

Module 4 — Natriuretic peptides (NICE-prioritised)

Measured values

NICE thresholds used: NT-proBNP <400 low; 400–2000 intermediate; ≥2000 high. BNP <100 low; 100–399 intermediate; ≥400 high.

Notes

AF context is taken from Module 2 rhythm selection (no duplicate AF checkbox).

If neither BNP nor NT-proBNP is available, NP tier is “unknown” and the tool will not output “supportive HFpEF” language or SGLT2 consideration text.

Safety: low NP reduces HF probability; echo surrogates should not override low NP within this framework.

Disclaimer & governance statement

Disclaimer: Intended to support clinical reasoning and documentation in suspected HF with EF ≥41%.

Does not replace specialist cardiology input and does not mandate treatment.

Clinical judgement, comorbidity, frailty, and local guidance must always be applied.

This framework does not dynamically adjust NP thresholds or echo interpretation for obesity, anaemia, lung disease or frailty/deconditioning. In such contexts, NP and resting echo may under- or over-estimate the contribution of HF.
